# Funny – Thème WordPress
 
Thème WordPress personnalisé réalisé dans le cadre du projet fil rouge conception wordpress.
 
## Description
Ce thème a été conçu pour le site de boisson énergisante Funny, dédiée à rebooster les jeunes et les sportifs.
 
## Fonctionnalités
- Design responsive (mobile & desktop)
- Personnalisation CSS selon la charte graphique
- Structure basée sur l’arborescence du projet
- Compatibilité avec l’éditeur de blocs WordPress
